# Transcript-Generator
Generates transcripts for Mumbai University Computer Engineering students.

### To Run
On cloning the repository, enter the command ```python generateTranscripts.py```. Transcripts will be generated in a folder called GeneratedTranscripts. <br/>
You can select the number of semesters you wish to generate transcripts for. Two sheets will be generated. Each sheet contains a maximum of four semesters.<br/> 
Please verify that the values that you have entered are correct. 

### On Generation of Transcripts
- ```Ctrl+A``` and set the font size to 8. 
- Add the letterhead on top.
- Copy this into a word document and submit the prints.
